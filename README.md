# Steps to replicate this template

Install https://github.com/rbenv/rbenv

Install a Ruby version, `rbenv install x.x.x`

Create a `.ruby-version` file specifying your Ruby version

Install https://bundler.io/ with `gem install bundler`

Configuring Bundler to install gems into a project `.gems` folder with `bundle install --path=.gems`

Create a `Gemfile`
